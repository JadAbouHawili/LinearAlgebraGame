1 Tutorial World

This world introduces basic concepts of Lean 4 and formal theorem proving. Players learn fundamental tactics and proof techniques through simple mathematical statements over 10 levels.

1.1 Level 1: The rfl Tactic

Lemma 1 Reflexivity of Equality
#

Lean declarations

  • TutorialWorld.level1

For any element \(x \in \mathbb {R}\), we have \(x = x\).

Proof ▶

This follows directly from the reflexivity of equality. In Lean, this is proven using the rfl tactic, which closes any goal where both sides are definitionally equal.

1.2 Level 2: The Rewrite Tactic

Lemma 2 Basic Substitution
#

Lean declarations

  • TutorialWorld.level2

Given \(y = x + 7\), prove that \(2y = 2(x + 7)\).

Proof ▶

Use the rw (rewrite) tactic to substitute the given equality, then apply reflexivity.

Lemma 3 Implication Reflexivity
#

Lean declarations

  • TutorialWorld.level3

For any proposition \(P\), we have \(P \rightarrow P\).

Proof ▶

Introduce the hypothesis using intro, then provide it exactly using exact.

Lemma 4 Conjunction Introduction
#

Lean declarations

  • TutorialWorld.level4

Given propositions \(P\) and \(Q\), if both hold, then \(P \land Q\) holds.

Proof ▶

Use the constructor tactic to split the conjunction goal into two separate goals.

1.5 Level 5: Unfold and Apply

Lemma 5 Contrapositive
#

Lean declarations

  • TutorialWorld.level5

For propositions \(P\) and \(Q\): \((P \rightarrow Q) \rightarrow (\neg Q \rightarrow \neg P)\).

Proof ▶

This classical result is proven by unfolding the definition of negation and applying the given implication.

1.6 Level 6: Cases and Contradiction

Lemma 6 Ex Falso Quodlibet
#

Lean declarations

  • TutorialWorld.level6

From a contradiction \(P \land \neg P\), any proposition \(Q\) follows.

Proof ▶

Use cases’ to decompose the conjunction, then exfalso to derive a contradiction.

Lemma 7 Inequality Addition
#

Lean declarations

  • TutorialWorld.level7

If \(x {\lt} y\) and \(a {\lt} b\), then \(x + a {\lt} y + b\).

Proof ▶

The linarith tactic solves this automatically by linear arithmetic reasoning.

Lemma 8 Group Simplification

Lean declarations

  • TutorialWorld.level8

In a group, \(a \cdot a^{-1} \cdot 1 \cdot b = b \cdot c \cdot c^{-1}\).

Proof ▶

The simp tactic simplifies using group axioms to establish the equality.

1.9 Level 9: Existential Quantifiers

Lemma 9 Existence Proof
#

Lean declarations

  • TutorialWorld.level9

For any real \(x\), we have \(x \leq 1 + x\).

Proof ▶

Use the use tactic to provide a witness for existential goals, establishing the inequality.

1.10 Level 10: Mathematical Induction

Lemma 10 Zero Addition
#

Lean declarations

  • TutorialWorld.level10

For any natural number \(n\), we have \(0 + n = n\).

Proof ▶

Proceed by induction on \(n\). The base case \(0 + 0 = 0\) follows from definitions. For the inductive step, assume \(0 + k = k\) and prove \(0 + (k+1) = k+1\) using the inductive hypothesis.

1.11 Summary of Tactics Learned

This world introduces the following essential Lean 4 tactics:

  • rfl - Reflexivity of equality

  • rw - Rewriting using equalities

  • intro - Introduction of assumptions

  • exact - Providing exact terms

  • constructor - Splitting constructive goals

  • unfold - Unfolding definitions

  • apply - Function application

  • cases’ - Case analysis

  • exfalso - Proof by contradiction

  • linarith - Linear arithmetic solver

  • simp - Simplification

  • use - Providing witnesses

  • induction’ - Mathematical induction
