3 Linear Independence and Span

This world explores the fundamental concepts of linear combinations, span, and linear independence through 10 challenging levels, culminating in sophisticated proofs about bases and spanning sets.

3.1 Linear Combinations and Span

Definition 18 Linear Combination
✓
#

Uses

  • Definition 11

Lean declarations

  • LinearAlgebraGame.is_linear_combination

A vector \(v \in V\) is a linear combination of a set \(S \subseteq V\) if there exist finitely many vectors \(v_1, \ldots , v_n \in S\) and scalars \(a_1, \ldots , a_n \in K\) such that:

\[ v = \sum _{i=1}^n a_i \cdot v_i \]
Definition 19 Span
✓
#

Uses

  • Definition 18

Lean declarations

  • LinearAlgebraGame.span

The span of a set \(S \subseteq V\), denoted \(\text{span}(S)\), is the set of all linear combinations of vectors from \(S\):

\[ \text{span}(S) = \{ v \in V : v \text{ is a linear combination of vectors in } S\} \]

3.2 Level 1: Introduction

This level introduces the concepts with a trivial proof to familiarize players with the notation and definitions.

3.3 Level 2: Elements are Linear Combinations

Theorem 20 Self Combination
#

Uses

  • Definition 18

Lean declarations

  • LinearAlgebraGame.linear_combination_of_mem

Every element of a set is a linear combination of that set:

\[ \forall v \in S: v \text{ is a linear combination of } S \]
Proof ▶

Take \(n = 1\), \(v_1 = v\), and \(a_1 = 1\). Then \(v = 1 \cdot v\) is a linear combination of \(S\).

3.4 Level 3: Elements Belong to Their Span

Theorem 21 Membership in Span
#

Uses

  • Definition 19
  • Theorem 20

Lean declarations

  • LinearAlgebraGame.mem_span_of_mem

Every element of a set belongs to its span:

\[ \forall v \in S: v \in \text{span}(S) \]
Proof ▶

By Level 2, \(v\) is a linear combination of \(S\), hence \(v \in \text{span}(S)\) by definition.

3.5 Level 4: Monotonicity of Span

Theorem 22 Span Monotonicity
#

Uses

  • Definition 19

Lean declarations

  • LinearAlgebraGame.span_mono

The span operation is monotonic:

\[ A \subseteq B \implies \text{span}(A) \subseteq \text{span}(B) \]
Proof ▶

Any linear combination of vectors from \(A\) is also a linear combination of vectors from \(B\), since \(A \subseteq B\).

3.6 Level 5: Linear Independence

Definition 23 Linear Independence
✓
#

Uses

  • Definition 18

Lean declarations

  • LinearAlgebraGame.linear_independent_v

A set \(S \subseteq V\) is linearly independent if the only way to express zero as a linear combination of distinct vectors from \(S\) is with all coefficients equal to zero.

Theorem 24 Empty Set Independence
#

Uses

  • Definition 23

Lean declarations

  • LinearAlgebraGame.linear_independent_empty

The empty set is linearly independent.

Proof ▶

There are no vectors to form a non-trivial linear combination, so the condition is vacuously satisfied.

3.7 Level 6: Independence of Subsets

Theorem 25 Subset Independence
#

Uses

  • Definition 23

Lean declarations

  • LinearAlgebraGame.subset_linear_independent

Subsets of linearly independent sets are linearly independent:

\[ A \text{ linearly independent} \land B \subseteq A \implies B \text{ linearly independent} \]
Proof ▶

Any linear dependence relation among vectors in \(B\) would also be a linear dependence relation among vectors in \(A\), contradicting the independence of \(A\).

3.8 Level 7: Supersets Span

Theorem 26 Superset Spanning
#

Uses

  • Definition 19
  • Theorem 22

Lean declarations

  • LinearAlgebraGame.superset_span_full

If \(A\) spans \(V\) and \(A \subseteq B\), then \(B\) spans \(V\):

\[ \text{span}(A) = V \land A \subseteq B \implies \text{span}(B) = V \]
Proof ▶

By monotonicity: \(V = \text{span}(A) \subseteq \text{span}(B) \subseteq V\), hence \(\text{span}(B) = V\).

3.9 Level 8: Uniqueness of Linear Combinations (Boss Level)

Theorem 27 Unique Representation
#

Uses

  • Definition 23
  • Definition 18

Lean declarations

  • LinearAlgebraGame.linear_combination_unique

If \(S\) is linearly independent, then every vector in \(\text{span}(S)\) has a unique representation as a linear combination of vectors from \(S\) (up to reordering and zero coefficients).

Proof ▶

Suppose \(v = \sum a_i v_i = \sum b_i v_i\) for distinct \(v_i \in S\). Then:

\[ 0 = \sum (a_i - b_i) v_i \]

By linear independence, \(a_i - b_i = 0\) for all \(i\), hence \(a_i = b_i\).

3.10 Level 9: Independence with Insertion (Challenge)

Theorem 28 Independence Extension
#

Uses

  • Definition 23
  • Definition 19

Lean declarations

  • LinearAlgebraGame.linear_independent_insert_of_not_in_span

If \(S\) is linearly independent and \(v \notin \text{span}(S)\), then \(S \cup \{ v\} \) is linearly independent.

Proof ▶

Suppose \(\sum a_i s_i + b \cdot v = 0\) where \(s_i \in S\) are distinct.

  • If \(b \neq 0\): Then \(v = -b^{-1} \sum a_i s_i \in \text{span}(S)\), contradiction.

  • If \(b = 0\): Then \(\sum a_i s_i = 0\), so \(a_i = 0\) for all \(i\) by independence of \(S\).

Therefore, all coefficients must be zero.

3.11 Level 10: Removing Redundant Vectors (Challenge)

Theorem 29 Redundant Vector Removal
#

Uses

  • Definition 19

Lean declarations

  • LinearAlgebraGame.remove_redundant_span

If \(w \in \text{span}(S \setminus \{ w\} )\), then:

\[ \text{span}(S) = \text{span}(S \setminus \{ w\} ) \]
Proof ▶

  • \(\text{span}(S \setminus \{ w\} ) \subseteq \text{span}(S)\): By monotonicity.

  • \(\text{span}(S) \subseteq \text{span}(S \setminus \{ w\} )\): Any linear combination involving \(w\) can be rewritten using the expression of \(w\) in terms of \(S \setminus \{ w\} \).

3.12 Summary

This world establishes the fundamental theory of linear independence and span:

  • The formal definitions of linear combinations and span

  • Key properties of the span operation (containment, monotonicity)

  • The concept of linear independence and its preservation under subset operations

  • Advanced results about unique representations and basis extension

  • The ability to remove redundant vectors from spanning sets

These concepts form the theoretical foundation for understanding bases, dimension, and the structure of vector spaces.
